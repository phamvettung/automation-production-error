# Phân loại sản phẩm lỗi trong dây chuyền tự động ứng dụng phương pháp học máy.

### CÁC PHƯƠNG PHÁP SỬ DỤNG
1. Phân loại sản phẩm sử dụng thuật toán k-nearest neighbors. Đánh giá mô hình với phương pháp xác thực chéo kết hợp lấy mẫu phân tầng.
2. Phân loại sản phẩm sử dụng thuật toán Support Vector Machines. Đánh giá mô hình với phương pháp lựa chọn mô hình sử dụng chiến lược Holdout
